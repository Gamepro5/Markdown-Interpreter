# Markdown Interpreter

A platform-agnostic markdown renderer that displays `.md` files with GitHub-style formatting and lets you download the result as a PDF.

## Usage

1. Open `index.html` in any modern browser
2. Drag & drop a markdown file onto the page (or click **Open File** to browse)
3. Click **Download PDF** to save the rendered output as a PDF

## Features

- **GitHub-flavored Markdown** — headings, bold, italic, links, images, blockquotes, tables, task lists, horizontal rules
- **Syntax-highlighted code blocks** — powered by highlight.js
- **PDF export** — one-click download with print-friendly light theme
- **Drag & drop** — drop a file anywhere on the page
- **Zero install** — single HTML file, no build step, no server required
